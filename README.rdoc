= Ping Pong Pros

This is our final project at Makers Academy, built in just ten days. With this project we hope to revolutionalise the world of Ping Pong / Table Tennis. You can view it here[http://www.pingpongpros.com/].

=== Technologies

- Rails 4
- Ruby 2.0
- PostgreSQL with ActiveRecord
- HTML5, HAML, CSS3, SASS
- CoffeeScript, JQuery and AJAX
- API Integration, JSON and JBuilder
- Ominauth for Facebook and Googlemaps
- Angular.js
- Devise
- Capybara-RSpec and Poltergeist
- Heroku
